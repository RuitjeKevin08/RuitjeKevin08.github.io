    • Mijn reis door een website coderen

    Hallo, mijn naam is Kevin Ruitenberg, en ik zit in havo 5 op het Vlietland College. Hier ga ik vertellen over mijn
    avontuur in het vak informatica, en wat ik van het vak en het proces vind. Informatica is namelijk de eerste keer
    dat ik ooit heb gecodeerd, en tot nu toe gaat het heel goed. Deze website heb ik helemaal zelf gemaakt, en ik vind
    hem goed gelukt, gezien dit de eerste keer is dat ik iets met websites maken heb gedaan, dit ging dus ook echt met
    vallen en opstaan. Belangrijk is ook dat je de code überhaupt snapt, want als je dat niet eens snapt, codeer je
    eigenlijk niet echt. Ik snapte er eerst ook helemaal niks van, maar nu wel, en kan ik (zonder de hele tijd te
    kopiëren en plakken) goede code schrijven. Als je mij tijdens de eerste les had gevraagd wat < !DOCTYPE html > doet,
    had ik het niet geweten. Nu kan ik echter vertellen dat de website zo kan zien dat het html code moet gebruiken om
    te werken, DOCTYPE staat ook letterlijk voor het type document dat moet worden gebruikt (DOCument TYPE)
